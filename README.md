# Docker 零基础 Web 部署指南

本项目提供了一份通俗易懂的 Docker 部署教程，旨在帮助新手快速掌握容器化技术的完整工作流，彻底解决“在我的电脑上明明能跑”的环境配置痛点。

教程以一个**校园二手交易平台 AI 估价 API** 为实战案例，演示了从零开始上云的全过程。

## 仓库文件介绍

本仓库包含该技术文档的两种格式，以满足不同的阅读和分发需求：

* **`部署Docker.md`**：纯文本格式的 Markdown 源码。适合直接在 GitHub 网页端阅读，包含了 Docker 核心概念（镜像、容器、仓库）的通俗解释、Dockerfile 编写实战以及常用部署命令。
* **`Docker部署.pdf`**：基于 XeLaTeX 编译导出的高质量 PDF 文档。拥有精美排版、标准目录结构和直观的架构图，非常适合作为技术附件随项目一同提交.

## 核心涵盖内容

1. **Docker 底层运转逻辑**：用“面向对象”思维拆解镜像 (Image)、容器 (Container) 与仓库 (Registry)。
2. **Dockerfile 编写规范**：以 Node.js 18-alpine 为基础镜像的实战配置，并包含利用缓存机制加速构建的技巧。
3. **本地构建与测试**：`docker build` 与 `docker run` 核心命令及端口映射详解。
4. **云端部署工作流**：将本地镜像推送至远端并拉取到云服务器运行的完整闭环。
